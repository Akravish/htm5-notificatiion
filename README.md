### Package manager

* [bower][4]

### Automatization

* [gulp][5]
* [BrowserSync] [6]

##  Preparing the environment

Install ruby packages

~~~bash
bundle install
~~~

Install node.js packages

~~~bash
npm install
~~~

Install bower packages

~~~bash
bower install
~~~

Run build + watch

~~~bash
gulp
~~~

### The default task
Starts:
* Compile the source code
* Start monitoring code
* Start BrowserSync
